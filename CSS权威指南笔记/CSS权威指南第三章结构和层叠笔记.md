  - 给定类属性值，属性选择或伪类，加0.0.1.0
  - 给定元素和伪元素，加0.0.0.1
  - 通配选择器，*{xxx}，0.0.0.0

  4. 按出现顺序排序，样式在文档中越后出现，权重越大。

  - 声明权重方面考虑5级，权重由大到小的顺序依次为：
    1. 读者的重要声明
    2. 创作人员的重要声明
    3. 创作人员的正常声明
    4. 读者的正常声明
    5. 用户代理声明
